# Search Engine

A simple Google-powered web search engine built with HTML, CSS, and JavaScript.

## Setup

1. Get a free Google API Key from [Google Cloud Console](https://console.cloud.google.com/)
2. Create a Custom Search Engine at [Programmable Search Engine](https://programmablesearchengine.google.com/)
3. Open `script.js` and replace the placeholders:
   ```js
   const API_KEY = "YOUR_GOOGLE_API_KEY";
   const CX = "YOUR_GOOGLE_CX";
   ```
4. Open `index.html` in your browser

## Technologies
- HTML / CSS / JavaScript
- Google Custom Search API
